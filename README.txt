Activation de l'environnement :

1) source botEnv/bin/activate
2) Installation de Python & Librairies

    - Python 3.6+ (https://www.python.org/downloads/)
            Librairies python :
                - Django (pip install django)
                - Librairie IBM watson (pip install --upgrade "ibm-watson>=4.4.0") 

Démarrage de l'outil de gestion :
3) Accéder au dossier contenant l'outil de gestion ( cd chatbot)
4) Démarrer l'application (python3 manage.py runserver)

Accès à la démo :
5) 127.0.0.1:8000/demo
